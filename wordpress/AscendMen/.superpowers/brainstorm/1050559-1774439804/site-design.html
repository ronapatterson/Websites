🏗 Architecture & Stack

Core
  • WordPress 6.x (installed via WP-CLI)
  • Kadence Theme (free)
  • Kadence Blocks Pro
  • PHP 8.1 / MySQL 8.0 / Apache
  • Served from /AscendMen/ locally
Key Plugins
  • Ultimate Member — accounts & membership
  • WooCommerce — paid programs
  • The Events Calendar — camp/event listings
  • Yoast SEO — search optimization
  • WP Mail SMTP — email delivery
  • UpdraftPlus — backups & migration
